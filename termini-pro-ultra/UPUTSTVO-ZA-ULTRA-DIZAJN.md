# Termini Pro Ultra dizajn

Ovo je nova, vizuelno jača verzija sajta.

## Kako da ubaciš na GitHub

1. Raspakuj ZIP.
2. Otvori GitHub repository `termini-pro`.
3. Klikni `Add file` pa `Upload files`.
4. Prevuci sve fajlove iz raspakovanog foldera.
5. Klikni `Commit changes`.

Render će automatski krenuti novi deploy. Ako ne krene sam, otvori Render servis i klikni `Manual Deploy` > `Deploy latest commit`.

## Najviše su promenjeni fajlovi

```txt
public/index.html
public/admin.html
public/css/styles.css
public/manifest.json
```
