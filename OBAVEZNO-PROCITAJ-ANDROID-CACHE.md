# Android i stari izgled

Ako se na Androidu ništa nije promenilo, telefon prikazuje staru PWA/cache verziju.

## Posle upload-a ove V2 verzije

1. Uploaduj sve fajlove na GitHub.
2. Sačekaj Render deploy da bude zelen/success.
3. Na telefonu otvori link ovako:

```txt
https://tvoj-link.onrender.com/?v=compact-v2
```

Ako i dalje vidiš staro:

## Chrome na Androidu

1. Otvori Chrome.
2. Idi na svoj sajt.
3. Klikni tri tačke gore desno.
4. Klikni `Site settings` ili `Podešavanja sajta`.
5. Klikni `Data stored` / `Storage`.
6. Klikni `Clear & reset` / `Clear data`.
7. Zatvori Chrome potpuno.
8. Otvori ponovo link:

```txt
https://tvoj-link.onrender.com/?v=compact-v2
```

## Ako si instalirao kao aplikaciju

Ako si ga dodao na početni ekran kao aplikaciju:

1. Drži ikonicu aplikacije.
2. Klikni `Uninstall` / `Remove`.
3. Otvori Chrome.
4. Otvori link sa `?v=compact-v2`.
5. Ponovo dodaj na početni ekran.

## Provera

Na kompjuteru otvori:

```txt
https://tvoj-link.onrender.com/?v=compact-v2
```

Ako kompjuter prikazuje novo, a telefon staro, problem je 100% keš na telefonu.
