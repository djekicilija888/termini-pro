# Owner Full Width Topbar v75

Header sada radi tačno ovako:

```txt
████████████████████████████████████████
PREMIUM                           Odjava
████████████████████████████████████████
```

Promene:

- crna traka ide od leve ivice ekrana do desne ivice ekrana
- širina je `100vw`
- nema margine levo/desno
- nema zaobljenih ivica
- nema ikonice/slova pored naziva firme
- naziv firme je levo
- `Odjava` je desno
- header je fiksiran gore dok se skroluje
- ostaje tako i kad se stranica zumira ili odzumira
