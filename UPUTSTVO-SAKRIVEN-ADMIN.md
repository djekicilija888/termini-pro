# Sakriven admin link

Ova verzija uklanja dugme `Admin` sa javne početne strane.

## Važno

Admin panel i dalje postoji, ali mušterije ne vide dugme.

Admin otvaraš ručno na:

```txt
https://termini-pro-2qyt.onrender.com/admin.html
```

## Posle upload-a na GitHub

Otvori javni sajt ovako:

```txt
https://termini-pro-2qyt.onrender.com/?v=no-admin-button-v1
```

## Ako Android prikazuje staro

Očisti cache:

```txt
Chrome → Settings → Site settings → All sites → termini-pro-2qyt.onrender.com → Clear & reset
```
