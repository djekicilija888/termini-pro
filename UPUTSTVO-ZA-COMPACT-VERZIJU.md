# Termini Pro Compact

Ovo je kraća i jednostavnija verzija sajta.

## Šta je popravljeno

- nema velikog skrolovanja
- forma je odmah vidljiva
- usluge su kratke i kompaktne
- telefon i kompjuter imaju isti stil
- na telefonu se elementi samo poređaju jedan ispod drugog
- nema ogromnih animacija i nepotrebnih sekcija

## Kako da ubaciš na postojeći javni sajt

1. Raspakuj ZIP.
2. Otvori GitHub repository `termini-pro`.
3. Klikni `Add file` → `Upload files`.
4. Prevuci sve fajlove iz raspakovanog foldera.
5. Klikni `Commit changes`.

Render će automatski uraditi novi deploy.

Ako ne krene sam:

1. Otvori Render.
2. Otvori servis `termini-pro`.
3. Klikni `Manual Deploy`.
4. Klikni `Deploy latest commit`.

## Najvažniji promenjeni fajlovi

```txt
public/index.html
public/css/styles.css
public/manifest.json
render.yaml
```
